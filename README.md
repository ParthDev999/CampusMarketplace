# GitHub Profile Analyzer

A modern GitHub analytics dashboard that allows users to search any GitHub profile and view useful developer insights such as profile statistics, repositories, language usage, achievements, activity score, and profile comparison.

## Live Demo

[View Live Project](https://campus-marketplace-frontend-eta.vercel.app/)

## Features

- Search any GitHub user profile
- View profile details such as avatar, bio, followers, following, and public repositories
- Display repositories with useful information
- Sort repositories by stars, forks, issues, size, name, and recent updates
- Analyze programming language usage
- Show developer scorecard and activity insights
- Compare GitHub profiles
- Dark mode support
- Clean and responsive user interface

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub API
- GitHub Pages for deployment

## Project Purpose

The purpose of this project is to provide a simple and useful dashboard for analyzing GitHub developers. It helps users quickly understand a developer's GitHub activity, repository quality, language distribution, and overall profile strength.
